# Shopping List - Flutter

Una aplicación de lista de compras desarrollada en Flutter con arquitectura MVVM.

## Características

- ✅ Autenticación con Firebase Auth
- ✅ Base de datos local SQLite con sqflite
- ✅ Sincronización con Firebase Firestore
- ✅ Arquitectura MVVM con Provider
- ✅ Gráficos de estadísticas
- ✅ Interfaz responsive
